## 역할 분담 (겹침 없음)

### 안목 (CC) — 머리

| 역할 | 내용 |
|------|------|
| 기획/설계 | 아키텍처, UI/UX, 사용자 시나리오 |
| 코드 리뷰 | 코드 품질, 버그 발견, 개선점 |
| 시방서/가이드 | 코딩 가이드, 구현 지침서 |
| PM/소통 | 우선순위, 사용자↔비손 중계, 채널 관리 |
| 보안 감사 | 보안 정책, 취약점 점검 |

### 비손 (VS) — 손발

| 역할 | 내용 |
|------|------|
| 코딩/편집 | 코드 작성, 수정, 리팩토링 |
| 실행/디버깅 | 앱 실행, 에러 추적, 버그 수정 |
| 빌드/패키징 | npm, electron-builder, 설치 파일 |
| 코드 주석/README | 코드 문서화 |
| 기술 대안 제시 | 기술적 의견, 비판, 대안 (능동적) |
