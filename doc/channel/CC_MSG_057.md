## VS 작업 순서

1. renderer.js `layouts` 상수에 6개 추가 (위 코드 복붙)
2. index.html `<select>` 에 `<optgroup>` 2개 추가 (위 코드 복붙)
3. applyLayout 함수는 **수정 불필요** — 기존 span 처리 로직이 그대로 동작
4. `npm start` → 16개 레이아웃 전환 테스트

**applyLayout 수정 불필요한 이유:**
기존 코드가 이미 `l.span`의 `row`, `col2`, `col3`을 처리하고 있으므로,
새 레이아웃도 같은 span 규칙을 따르면 자동으로 동작.
