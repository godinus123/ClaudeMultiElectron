주요 기능

  • 멀티패널 — claude.ai를 1~9개 동시 사용, 각각 독립 대화
  • 레이아웃 9종 — 1전체, 2좌우, 2상하, 4쿼드, 6그리드, 9채널, 비대칭 3종
  • 패널 스왑 — 뱃지/닉네임 클릭 → 액티브 위치로 즉시 교체
  • 브로드캐스트 — 하단 입력창에서 전체 또는 특정 패널로 메시지 전송
  • 클립보드 — [잡기]로 응답 복사 → [놓기]로 다른 패널에 전달
  • 응답 전달 — 드롭다운(➡)으로 패널 간 응답 릴레이
  • 드래그앤드롭 — 텍스트/파일을 패널 위에 드롭하여 전달
  • 설정 — ⚙ 버튼으로 패널 닉네임/색상 변경
  • 주소창 — URL 입력 + Enter로 이동 (화이트리스트 검증)
  • 로그인 공유 — 한 번 로그인하면 모든 패널에 자동 적용

단축키

  • ESC — 이 창 닫기

보안

  • 소스코드 100% 공개 (GitHub)
  • API 키/토큰 없음 — claude.ai 웹 로그인 사용
  • HTTPS 전용, URL 화이트리스트 적용
  • 세션 데이터는 로컬 PC에만 저장 (%APPDATA%)
